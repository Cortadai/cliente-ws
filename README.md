Este repositorio es un **cliente de servicios web SOAP** desarrollado con Spring Boot 2.7.3 y Java 11. El proyecto implementa dos clientes SOAP que consumen servicios web externos: uno para consultar información de países (PaisClient) que incluye datos como nombre, capital, población y moneda, y otro para consultar información de productos (ProductosClient) con detalles como nombre, descripción, tipo y estado. La aplicación utiliza JAXB para el marshalling/unmarshalling de XML, configura marshallers separados para cada servicio mediante Spring Web Services, y al ejecutarse como CommandLineRunner realiza consultas de demostración a los servicios (esperando que estén corriendo en localhost:8080 y localhost:8082), imprimiendo los resultados en consola. Es un proyecto de ejemplo educativo que demuestra cómo consumir servicios SOAP basados en WSDL usando el ecosistema Spring.
